功能特色

为阅读外文网页而生

每一项功能都围绕「快速、清晰、可控」的翻译体验打磨。

一键全文翻译

点击即可翻译整个页面，无需选中文字。整页内容即刻获得对照译文。

双语对照渲染

保留原文，译文紧跟其后插入，对照阅读更清晰，长文也不迷路。

多翻译服务支持

内置 Google Web、微软、Google Cloud、OpenAI 兼容、HTTP 模板，按需切换。

本地缓存

按「服务+目标语言+原文」缓存，重复内容不重复请求，省额度也省时间。

悬浮球快捷入口

页面悬浮球，随时唤起翻译，无需打开弹窗，操作路径更短。

段落模式

可切换段落级翻译，精细化控制翻译粒度，适合逐段精读。

功能指导

几步开始使用

从安装扩展到高级配置，跟着步骤即可跑通完整翻译流程。

  1. 01

    安装扩展

    从 dist/ 目录在 chrome://extensions 或 edge://extensions 开启开发者模式后加载。

  2. 02

    打开弹窗

    点击浏览器工具栏的 WuPage 图标，呼出翻译控制面板。

  3. 03

    选择目标语言

    在「目标语言」下拉中选择要翻译成的语言。

  4. 04

    选择翻译服务

    在「翻译服务」下拉中选择服务商（推荐先用免 Key 的 Google Web）。

  5. 05

    点击「翻译全文」

    一键翻译，原文下方即出现译文，双语对照即刻呈现。

  6. 06

    高级设置

    点击「设置」进入选项页，可配置分块大小、并发数、缓存、各服务商密钥。

  7. 07

    清除缓存

    弹窗内点「清除缓存」可重置本地翻译缓存，便于调试或更换服务。

The State of Web Translation in 2026

The quick brown fox jumps over the lazy dog. 敏捷的棕色狐狸跳过了懒狗。

Modern browsers allow extensions to access page content directly, making real-time bilingual reading a reality. 现代浏览器允许扩展直接访问页面内容，让实时双语阅读成为现实。

API 获取流程

五种翻译服务接入指南

从免 Key 开箱即用到完全自定义模板，总有一种适合你。

免 Key

Google Web Translate

无需 API Key，开箱即用，适合快速体验。

端点 https://translate.googleapis.com/translate_a/single
非官方接口，可能被限速、屏蔽或变更。
  1. 1

    直接选择服务

    在 WuPage 弹窗的「翻译服务」下拉中选择「Google Web Translate」即可。

  2. 2

    无需任何配置

    无需填写 Key、Endpoint，立即点击「翻译全文」开始使用。

Azure

Microsoft Translator

微软 Azure 认知服务翻译，稳定、支持区域路由。

端点 POST https://api.cognitive.microsofttranslator.com/translate?api-version=3.0&to={targetLang}
  1. 1

    创建 Translator 资源

    登录 Azure Portal，创建「Translator」资源。

  2. 2

    获取 Key 与区域

    在资源的「密钥和终结点」页获取 Key 和区域（如 eastasia）。

  3. 3

    填入 WuPage

    在 WuPage 选项页填入 API key 与 Region，保存后即可使用。

GCP

Google Cloud Translation

官方 Translation Basic API，按量计费、稳定可靠。

端点 POST https://translation.googleapis.com/language/translate/v2?key={apiKey}
  1. 1

    启用 API

    登录 Google Cloud Console，启用 Translation Basic API。

  2. 2

    创建 API Key

    在「凭据」页面创建 API Key，建议限制其只调用 Translation API。

  3. 3

    填入 WuPage

    在 WuPage 选项页填入 API key，保存即可。

LLM

OpenAI 兼容

支持任意 OpenAI 兼容的 /chat/completions 服务，如 OpenAI、DeepSeek 等。

示例 POST {baseURL}/chat/completions
  1. 1

    获取凭据

    在服务商处获取 API Key 与 baseURL。

  2. 2

    填入配置

    在 WuPage 选项页填入 baseURL、API key、model、prompt。

自定义

HTTP 模板

适配任意免费 / 付费翻译 API，按模板拼装请求与解析响应。

支持占位符
{{targetLang}}{{sourceLang}}{{texts}}{{json texts}}
响应路径必须解析为与输入顺序一致的译文数组。
  1. 1

    填写请求模板

    在选项页填入 URL 模板、请求方法（GET/POST）、请求体模板。

  2. 2

    填写响应路径

    配置响应路径，使解析结果为顺序一致的译文数组。

  3. 3

    测试并保存

    点击「测试服务」验证后保存，即可在弹窗选择该服务使用。

下载安装

手动加载扩展

当前为开发者预览版本，按以下步骤在 Chrome 或 Edge 中手动加载。

1

克隆仓库

git clone https://github.com/mr-wuliu/wupage
2

安装依赖并构建

npm install && npm run build
3

打开扩展管理页

Chrome 访问 chrome://extensions，Edge 访问 edge://extensions。

4

开启「开发者模式」

在扩展管理页右上角勾选「开发者模式」开关。

5

加载已解压的扩展程序

点击「加载已解压的扩展程序」，选择仓库中的 dist/ 目录。
